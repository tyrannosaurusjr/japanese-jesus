# Editorial Boundaries

This is the operating rule for public Japanese Jesus content.

## Safe Pattern

Use a three-layer model:

1. Fact
2. Interpretation
3. Canon

### Fact

Facts must be:

- real
- attributable
- sourceable
- stated plainly

Examples:

- Shingo population
- location
- transport details
- festival information
- municipal facilities

### Interpretation

Interpretation can be written in the Japanese Jesus voice, but it must still read as interpretation.

Good framing:

- "In canon terms..."
- "As a conduit reading..."
- "The site treats this as..."
- "The pattern suggests..."

Bad framing:

- invented claims presented as municipal fact
- pseudo-statistical statements presented as established truth
- myth statements attributed to civic or official sources

### Canon

Canon can be fully in-world, but it belongs in clearly mythic containers:

- Canon pages
- Conduit readings
- Signal analysis
- Relic descriptions

It should not be disguised as travel guidance or civic reference copy.

## Public Content Rule

If a sentence could be mistaken for a real-world factual claim, it needs either:

- a real source, or
- clear framing that it is interpretation

This keeps the site strange without making it misleading.
