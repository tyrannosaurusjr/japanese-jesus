# Curated References

This is the working source list for public-facing Japanese Jesus pages.

Use primary and official sources first. Use secondary sources for orientation only.

## Primary / Official

### Shingo Village Municipal Site

- URL: https://www.vill.shingo.aomori.jp/
- Use for: municipal facts, administrative references, local framing, baseline civic data
- Note: if population figures differ across sources, prefer the municipal site for displayed factual copy

### Amazing AOMORI (Official Aomori Travel Guide)

- URL: https://aomori-tourism.com/en/spot/detail_62.html

- Note: useful as a practical travel source for the Conduit and Journey pages

### Aomori Airport Official Site

- Use for: airport access, flight status, practical journey references

### JR East Tohoku Shinkansen Route Map

## Suggested On-Site Usage

- Canon: reference only when grounding chronology or geography
- Conduit: use official Shingo and Aomori tourism sources, then add one or two secondary confirmations
- Journey: use airport, JR East, and lodging/travel partners
- Signal: reference only for content promises, not for source-heavy pages

## Editorial Rule

Facts should stay factual.

Canon extrapolations should be clearly written as in-world interpretation layered on top of public information, not disguised as civic data.
