The goal is to finish the canon and conduit hubs before expanding hidden ARG layers.

## Conduit Structure

The Conduit page is the Shingo / Herai hub.

Field Notes should be treated as posts, not static card blurbs.

Minimum target:

- 800 to 1,600 words per field note
- add local references
- add outbound links
- add supporting image(s)
- add one factual layer and one canon extrapolation layer

Initial note sequence:

1. Primary Node
2. Conduit Signals
3. Seasonal Residue
4. The Northern Frontier

Additional note candidates:

1. Population Signal
2. Civic Thresholds
3. Christ Park / tourism infrastructure
4. Weather windows
5. Access logic from Hachinohe

## Homepage Priorities

The homepage should function as a true hub index:

- clear route into Canon
- clear route into Conduit
- clear route into Journey
- clear route into Relics
- clear route into Signal

It should not try to carry the full content burden itself.

## Out Of Scope For This Phase

- hidden ARG depth
- puzzle-specific copy
- cryptic micro-layers that interfere with public clarity

Finish the public narrative system first.
